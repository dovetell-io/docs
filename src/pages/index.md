operating knowledge across tools and sessions.

If you are an agent reading this because a human pointed you here, start with
the [Agent start checklist](/guides/agent-start-checklist). Your job is to
orient, make the smallest useful context setup, show the human what changed, and
avoid turning their repo into a process museum.

The docs are early and intentionally lightweight. Public pages explain stable
concepts and reviewed guidance. Template source and paid package material are
kept separate until promoted.
