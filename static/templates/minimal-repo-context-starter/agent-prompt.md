# Agent Prompt

Use this prompt after copying the starter into a repo.

```text
Read this repository and adapt the minimal repo context starter.

Start by reading README.md, AGENTS.md if present, and any existing context
folder. Then update only the copied starter files needed to make the next work
session easier to resume.

Requirements:
- identify the repo role
- preserve the repo's actual source-of-truth boundaries
- remove sample-only language
- do not add private strategy to a public repo
- keep tasks active and specific
- record only durable decisions that future sessions need
- keep the setup small

Finish by telling me:
- what repo role you found
- what files you changed
- what I should review
- what you intentionally did not add
```
