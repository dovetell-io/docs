The Team AI Maturity Model describes how systematically a team uses AI.

Every team is somewhere on this curve. dovetell helps the team move up it by
making shared context, review, and reuse more deliberate.

## Levels

| Level | Score | Pattern |
| --- | ---: | --- |
| Scattered | 0-22 / 90 | AI use is isolated. Context lives in individual chats. No shared standards or reuse across the team. |
| Structured | 23-54 / 90 | Some repeatable practices exist, but they are inconsistent. Context is scattered and handoffs vary by person. |
| Coordinated | 55-72 / 90 | The team has shared context and standard prompts. AI-assisted work is more consistent across people. |
| Compounding | 73-90 / 90 | AI context compounds over time. Decisions become searchable precedent and the system improves itself. |

## Six capability areas

Teams are scored across six dimensions. Each dimension has five questions.
Each question is scored from 0 to 3.

| Area | What it measures |
| --- | --- |
| Shared context | Does team-wide context exist and is it discoverable? |
| Prompt reuse | Are high-value prompts captured and shared? |
| Team handoffs | Is there a standard handoff format with clear ownership? |
| Knowledge capture | Are insights and decisions captured systematically? |
| Review and governance | Is there a review process for AI outputs and decisions? |
| Workflow integration | Is AI integrated consistently across the whole team? |

Thresholds are current estimates. They should be recalibrated after enough
non-founder completions create a useful score distribution.
