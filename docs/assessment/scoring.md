---
title: Scoring
description: How the dovetell assessment calculates points and assigns maturity levels.
---

The assessment uses a simple point model.

```text
6 sections
x 5 questions per section
x max 3 points per question
= 90 points maximum
```

Unanswered or skipped questions score 0.

## Answer scale

| Score | Label |
| ---: | --- |
| 0 | Not true |
| 1 | Somewhat true |
| 2 | Mostly true |
| 3 | Very true |

## Level thresholds

| Level | Score | Percent of max |
| --- | ---: | ---: |
| Scattered | 0-22 | 0-24% |
| Structured | 23-54 | 26-60% |
| Coordinated | 55-72 | 61-80% |
| Compounding | 73-90 | 81-100% |

## Calibration

These thresholds are working estimates.

They should be recalibrated when there are enough real, non-founder completions
to see the score distribution. Until then, the score is best used as a
directional guide, not a scientific benchmark.
