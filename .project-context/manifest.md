## Purpose

This repo hosts public dovetell documentation at `docs.dovetell.io`.

It should explain stable public concepts, guides, template overviews, and
reference material. It should not store private strategy, customer details,
credentials, internal launch notes, or unreviewed source package material.

- Public docs should be promoted from reviewed source material, not copied from
  private context wholesale.
